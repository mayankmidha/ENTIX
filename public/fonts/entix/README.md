Entix brand fonts
==================

Loaded in production:
- Body: Glacial Indifference Regular / Bold, SIL Open Font License 1.1.
- Sub heading: Times New Roman MT, with Times New Roman as the system fallback.
- Heading: Brown Sugar via local/purchased font slot and Entix vector logo assets.

Font files included now:
- Glacial Indifference Regular / Italic / Bold, SIL Open Font License 1.1.
- Times New Roman MT falls back to the system Times New Roman family.

Pending licensed webfont:
- Brown Sugar by EmbunStudio / Aditya Rezki Apriyadi.

Brown Sugar is named in the Entix PDF and used in the vector logo assets, but public font archives list the demo as personal-use only. The matching Creative Fabrica listing is approximately $2.70 on sale / $9.00 regular at the time checked, with a commercial license. For client launch, place the purchased webfont here as `BrownSugar.woff2` or `BrownSugar.woff`, then add it to the Brown Sugar `@font-face` rule in `src/app/globals.css`.
